# ReactJS Store-app #
--------

Проект скопилирован и готов к запуску:
`/public/index.html`

### Запуск DEV-сервера ###

1. Закрузить проект на ПК
2. `cd react-shop-master`
3. `npm install`
4. Run server >> `npm run devserver`
5. **[http://localhost:7878/](http://localhost:7878/)**
